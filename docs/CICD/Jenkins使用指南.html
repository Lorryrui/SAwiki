Jenkins使用指南

一、 Jenkins分布式构建

​ Jenkins使用主从架构管理分布式构建。主节点处理构建调度工作，分发到不同的从节点处理。Jenkins主实例也可以直接执行构建作业。

1.1 从节点配置

​ 1.1.1 安装java环境并添加jenkins用户

yum -y install java-1.8.0-openjdk.x86_64
useradd jenkins

​ 1.1.2 生成公私钥并添加主实例的公钥

ssh-keygen

​ 在/home/jenkins/.ssh目录下添加主实例的authorized_keys文件

1.2 主节点配置

​ Jenkins管理后台，Jenkins-->管理节点-->新建节点

​ ① 远程工作目录配置从节点的工作目录，配置jenkins用户的家目录

​ ② 启动方法是主实例连接从节点的方式，这里使用的是SSH

​ ③ Host是配置的从节点的IP地址

​ ④ Credentials配置的是主实例使用证书连接从节点的SSH用户名和主实例的私钥
